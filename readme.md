# GitHub Fonts Superfamily Plugin for Acode

A premium, modern typographical extension for the Acode Mobile Editor. This plugin brings GitHub's complete high-performance **Monaspace font superfamily** (Neon, Radon, Xenon, Krypton, and Argon) directly into your mobile development workflow with a beautiful custom sidebar UI management panel.

## 🌟 Features

- **Full Monaspace Superfamily Support:** Includes all 5 distinct monospaced font families engineered by GitHub:
  - **Neon:** Modern, clean, and highly legible neo-grotesque variant.
  - **Radon:** Stylish, cursive-influenced aesthetic for script-lovers.
  - **Xenon:** Classic, high-contrast serif structure for structured layout reading.
  - **Krypton:** Bold, industrial, technological block terminal look.
  - **Argon:** Sophisticated, wide geometric humanistic design structure.
- **Comprehensive Weight Ranges:** Native bundling of Regular (400), Medium (500), SemiBold (600), and Bold (700) weights per family.
